HOW TO GENERATE MULTIPLE EXPERIMENTS:

  1 -   GENERATE A.OUT FOM BUCLE.C
        @:~$ gcc bucle.c
        
  2 -   TO PERFORM 10 EXPERIMENTS
        @:~$ ./a.out 10
  
  3 -   timeM.log & timeQ.log CONTAIN ALL DATA 
        (30 TESTS EACH).
        LOG_M.log & LOG_Q.log CONTAIN BRIEFED DATA 
        (10 TESTS EACH).
     
  4 -   DELETING EVERYTHING BUT THE ESSENTIAL,
        SAVE LOGS BEFORE NEXT COMMAND.
        @:~$ make rm
